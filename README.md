#  Playwright (TypeScript)

##  Overview

This project is an automated end-to-end testing suite built using **Playwright with TypeScript**.
It validates core functionalities of the Swag Labs demo application:
 https://www.saucedemo.com/

The project demonstrates modern UI test automation practices including structured test design, assertions, and handling real user workflows.

---

##  Features Covered

###  1. Login Functionality

* Navigate to Swag Labs website
* Login using valid credentials
* Validate successful navigation to products page
* Capture screenshot after login

---

### 2. Product Sorting

* Select sorting option: **Price (low to high)**
* Verify products are sorted correctly
* Validate product titles and prices

---

###  3. Cart Functionality

* Add multiple products to cart
* Verify cart badge count
* Remove items from cart
* Validate updated cart contents

---

###  4. Checkout Workflow

* Add items to cart
* Proceed to checkout
* Fill in user details (First Name, Last Name, Postal Code)
* Validate total price and tax
* Complete checkout process
* Verify successful order confirmation

---

###  5. Negative Test Scenarios

* Invalid login credentials → error validation
* Checkout with missing fields → validation errors

## ⚙️ Setup & Installation

### 1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/qa-playwright-assessment.git
cd qa-playwright-assessment
```

### 2. Install dependencies

```
npm install
```

### 3. Install Playwright browsers

```
npx playwright install
```

---

##  Run Tests

### Run all tests

```
npx playwright test
```

### Run tests in headed mode

```
npx playwright test --headed
```

### Show test report

```
npx playwright show-report
```
